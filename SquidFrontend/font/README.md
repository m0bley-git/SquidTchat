# Font

Ce dossier contient les polices d'écriture utilisées dans l'interface de SquidTchat.

## Police utilisée

**Aristotelica** — AristotelicaSmallCaps-Regular.ttf

## Utilisation
```css
@font-face {
    font-family: 'Aristotelica';
    src: url('../font/aristotelica/AristotelicaSmallCaps-Regular.ttf');
}
```
